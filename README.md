# JesseRai Salon

A modern, elegant website for JesseRai Salon in Annapolis, Maryland.

## About

JesseRai is a premium hair salon offering high-quality, affordable services to the Annapolis community. Located off Route 2 near Harbor Center, we provide exceptional hair care in a welcoming atmosphere.

**Owner:** Jessica Angell

## Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** CSS Modules
- **Fonts:** Lora & Zeyada (Google Fonts)
- **Deployment:** Vercel

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Accessibility

This site follows WCAG 2.1 guidelines and includes:

- Semantic HTML
- Proper heading hierarchy
- Focus visible indicators
- Skip to content link
- Responsive design
- Reduced motion support
